# Tech Icons Directory

Place your technology icon images here. Each image should be a PNG or SVG file.

Required files:
- javascript.png
- typescript.png
- react.png
- nextjs.png
- tailwind.png
- mui.png
- bootstrap.png
- shadcn.png
- nodejs.png
- express.png

Recommended size: 64x64px or 128x128px for best quality.

You can download icons from:
- https://devicon.dev/
- https://simpleicons.org/
- https://www.svgrepo.com/
